# Instructions: How to Join the Matrix Space

Please follow the steps below to join the Matrix space for this VU using the Element web client.

## **Important**: Home Server Selection

* The correct **home server** must be selected, otherwise you will **not be able to join the space**.
* Students should use:
    * **Students with accounts like `xxx@student.uibk.ac.at`**:
        * Home server: **`matrix2.uibk.ac.at`**

    * **University work accounts like `xxx@uibk.ac.at`** (not student accounts):
        * Home server: **`matrix1.uibk.ac.at`**

---

## If You are Already Logged in to Element

1. Open **Element in your browser**.
2. Make sure you are logged in using the correct home server (see above).
3. Click the invitation link provided.
4. If you see the message:
   *“Element is open in another window. Click ‘Continue’ to use Element here and disconnect the other window.”*  
   → Click **Continue**.
5. You should now see the space.
6. Click **Join**.

You have now successfully joined the space.

---

## If You are NOT Logged in to Element

1. Click the invitation link.
2. You will be redirected to the Matrix sign-in page.
3. Select the correct **Home Server** first (very important).
4. Click **Continue with SAML**.
5. Log in using your university credentials.
6. Click **Continue to your account**, then click **Continue**.
7. If a **“Confirm Identity”** window appears, you may verify your device if you want to. However, verification is **not required** to join the space.
   * If you prefer not to verify now, simply close the window using the **X** in the top-right corner and select **I’ll verify later**.
8. You should now see the space.
9. Click **Join**.

You have now successfully joined the space.

---

## Troubleshooting

* If the matrix space is not displayed after login, close the current browser tab and start the process again from the beginning.
* The system can occasionally behave inconsistently, so restarting the process usually resolves the issue.
